This project is a room sharing web platform developed as a graduation project.

## Project Overview

The platform allows users to:
- View available shared rooms
- Book rooms based on availability
- Enforces room capacity rules on the backend

The system is designed with real-world backend logic, including proper validation and error handling.

## Tech Stack

- Backend: Go (net/http)
- Database: PostgreSQL
- API Testing: Postman
- Version Control: Git & GitHub

## Current Status

- Database schema implemented
- Booking creation logic completed
- Capacity validation enforced
- API tested using Postman
